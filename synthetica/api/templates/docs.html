API Documentation

Build powerful synthetic data pipelines with the Synthetica API

Introduction

The Synthetica API allows you to programmatically generate high-quality synthetic customer support conversations for training AI models. All responses are returned in JSON format.

Base URL: http://localhost:8000

Authentication

All API requests require an API key to be included in the request header. You can obtain a free API key by creating an account on the pricing page.

HTTP Header 
X-API-Key: your-api-key-here
Security Note: Never share your API key publicly or commit it to version control. Store it securely as an environment variable.

Rate Limits

Rate limits are enforced based on your subscription tier. Each API call counts towards your monthly limit.

Tier Monthly Limit Price
Free 100 conversations $0
Starter 1,000 conversations $9
Growth 10,000 conversations $49

Generate Conversations

Generate synthetic customer support conversations based on your specifications.

Endpoint 
POST /api/generate

Headers

Header Value
X-API-Key Your API key
Content-Type application/json

Request Body

Parameter Type Required Description
industry string Required Industry context (e.g., "technology", "e-commerce")
topics array Required List of conversation topics (min 1 item)
count integer Optional Number of conversations (1-50, default: 1)
customer_tone string Optional Customer tone: "professional", "casual", "formal", "empathetic"
agent_style string Optional Agent communication style
message_count integer Optional Messages per conversation (4-20, default: 8)

{

Python Example

Complete example using Python and the requests library.

Python 
import requests
import os

# Configuration
API_KEY = os.getenv("SYNTHETICA_API_KEY")
BASE_URL = "http://localhost:8000"

# Generate conversations
response = requests.post(
    f"{BASE_URL}/api/generate",
    headers={
        "X-API-Key": API_KEY,
        "Content-Type": "application/json"
    },
    json={
        "industry": "technology",
        "topics": ["software bug", "account access"],
        "count": 10,
        "customer_tone": "professional",
        "message_count": 8
    }
)

if response.status_code == 200:
    result = response.json()
    print(f"Generated {result['count']} conversations")
    print(f"Average quality: {result['average_quality']}")

    # Access conversations
    for conv in result['conversations']:
        print(f"\nConversation ID: {conv['id']}")
        print(f"Quality Score: {conv['quality_score']}")
        print(f"Metrics: {conv['quality_metrics']}")
else:
    print(f"Error: {response.json()}")

Node.js Example

Complete example using Node.js and axios.

JavaScript 
const axios = require('axios');

const API_KEY = process.env.SYNTHETICA_API_KEY;
const BASE_URL = 'http://localhost:8000';

async function generateConversations() {
  try {
    const response = await axios.post(
      `${BASE_URL}/api/generate`,
      {
        industry: 'technology',
        topics: ['software bug', 'account access'],
        count: 10,
        customer_tone: 'professional',
        message_count: 8
      },
      {
        headers: {
          'X-API-Key': API_KEY,
          'Content-Type': 'application/json'
        }
      }
    );

    const result = response.data;
    console.log(`Generated ${result.count} conversations`);
    console.log(`Average quality: ${result.average_quality}`);

    // Access conversations
    result.conversations.forEach(conv => {
      console.log(`\nConversation ID: ${conv.id}`);
      console.log(`Quality Score: ${conv.quality_score}`);
      console.log(`Metrics:`, conv.quality_metrics);
    });
  } catch (error) {
    console.error('Error:', error.response?.data || error.message);
  }
}

generateConversations();
